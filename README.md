# Ben Calculator

![no text](IMG/demo.png "no title")

## Description
Shows Bens working time in minutes and seconds. Written is simply Java.

## Install

### With makefile

```text
make
make run
```

### Manual

Compiling

```text
javac src/Gui.java -d ""
javac src/Main.java -d ""
```

Running
```text
java benCalc.Main
```
